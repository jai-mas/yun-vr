# yun-vr

云VR项目 - 团队实践

## 项目介绍

这是一个基于uni-app开发的云VR项目，提供虚拟现实相关的功能和服务。

## 技术栈

- Vue.js 2.x
- uni-app
- uView UI

## 功能模块

- 首页
- 导航
- 探索发现
- 地图
- 展览
- 我的创作
- 个人中心
- 福利活动

## 安装与运行

```bash
# 安装依赖
npm install

# 运行开发环境
npm run dev:mp-weixin

# 打包
npm run build:mp-weixin
```

## 目录结构

- api: 接口请求
- common: 公共工具和配置
- compents: 自定义组件
- pages: 页面文件
- static: 静态资源
- uni_modules: uni-app模块
